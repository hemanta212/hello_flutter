# Learn Flutter

website: https://flutter.dev
docs: https://flutter.dev/docs

## Project Installation:
```
$ git clone https://github.com/hemanta212/hello_flutter.git
$ cd hello_flutter
$ flutter upgrade
```

There are many projects navigate to one and run.
```
$ flutter pub get
$ flutter clean
$ flutter run
```
